# 배포 전 5분 체크리스트

실무 반영 전 **Preview URL**에서 돌리는 수동 점검입니다.  
체크 하나라도 실패하면 **Production 배포 보류** → 수정 후 Preview 재확인.

---

## 0. 배포 직전 (30초)

- [ ] 로컬: `cd web && npm run build` 성공
- [ ] 이번 배포에서 바꾼 화면/기능을 한 줄로 메모 (롤백·원인 추적용)
- [ ] Vercel **Preview** URL 열기 (Production 아님)

**데모 계정·역할:** 로그인 후 상단 **역할 전환** 사용  
| 역할 | 용도 |
|------|------|
| 담당 (staff) | 실행 진행 · 계약 |
| 파트너 (partner) | 파트너 포털 |
| 고객사 (client) | (선택) 고객사 포털 |

**추천 테스트 계약:** `제주 오성` (`c-1`) — 데이터가 풍부함

---

## 1. 실행 진행 `/executions` (약 2분) · 담당 역할

### 1-1. 기본 로딩
- [ ] 페이지 오류 없이 열림
- [ ] 계약 업체 선택 · 검색 동작
- [ ] **조회 기간**: 기본 `계약` · 일/월/년/기간 전환 시 KPI·타임라인 숫자 급변/빈 화면 없음
- [ ] **고객사 포털 대시보드** 버튼 → `/contracts/{id}/client-portal` 이동

### 1-2. KPI · 집행
- [ ] 채널 KPI 카드: **월광고비·입금일·성과급 날짜 없음** (종합 달성 + 채널별 색상만)
- [ ] **집행 실행 · 채널별** 목록 표시 (추가/수정 모달 열림)

### 1-3. 업무 타임라인 (건당)
- [ ] 타임라인 카드: **노출채널 · 키워드 · 피드백 링크(회차 1건)** 표시
- [ ] `draft`/`rejected` 건: **파트너·비용 설정** 모달 열림
  - [ ] 비용 있음 → 해당 분야 파트너만
  - [ ] **비용 없음** 체크 → 전체 파트너(기자단·인플루언서 등) 표시
- [ ] `delivered` 건: **입금 확인** / 비용 0이면 **완료 확인** 버튼 동작(에러 alert 없음)

### 1-4. 입금 차단 (해당 계약만)
- [ ] 입금 미확인 계약 선택 시 업무 버튼 비활성 + 안내 문구 표시

---

## 2. 계약 상세 `/contracts/{id}` (약 1분 30초) · 담당 역할

**예:** `/contracts/c-1`

- [ ] KPI 스트립: 실행 진행과 **동일 규칙** (채널 색상, 월광고비/입금일/성과급 없음)
- [ ] **고객사 포털** 버튼 동작
- [ ] 탭 전환(개요·집행·연장 등) 시 크래시 없음
- [ ] 집행 실행 편집 → 포스팅 링크 저장 → 목록 반영
- [ ] (최근 수정 시) **재계약 조건 변경** · 팀장 결재 흐름 한 번 확인

---

## 3. 파트너 포털 `/dashboard` (약 1분 30초) · 파트너 역할

역할 전환 → **파트ner** → 대시보드

### 3-1. 승인 · 납품
- [ ] **승인 대기** 탭: 업무 카드 표시 · 승인/반려 동작
- [ ] **진행 중** 탭: 승인된 업무 · **결과 URL 제출** (`PartnerWorkDeliverCard`) 동작

### 3-2. 연동 확인 (담당으로 돌아가기)
- [ ] 역할 → **담당** → `/executions` 동일 계약
- [ ] 파트너가 제출한 링크가 타임라인 **피드백 링크**에 회차 1건으로 보임

---

## 4. Production 배포 & 사고 시

### 배포
- [ ] Preview 전 항목 통과
- [ ] Production 배포
- [ ] 배포 직후 Production에서 **1-1, 2 KPI, 3-1**만 1분 재확인

### 버그 발생 시
1. Vercel → 해당 배포 → **Rollback** (이전 버전)
2. Preview에서 재현 → 원인 커밋 특정
3. 수정 → Preview 재체크 → 재배포
4. 체크리스트에 **빠진 케이스 1줄** 추가 (다음 배포부터)

## 빠른 참고 · 자주 깨지는 지점

| 증상 | 의심 구간 |
|------|-----------|
| 타임라인 링크 5건 한꺼번에 | 회차 매칭 · `resolveWorkOrderPostLink` |
| KPI 숫자와 타임라인 건수 불일치 | 조회 기간 필터 |
| 입금 확인 후 집행 미반영 | `applyConfirmWorkOrderPayment` · 오더준 |
| 파트너 목록 비어 있음 | 비용 없음/있음 · 분야 필터 |
| 고객사만 화면 깨짐 | `/contracts/{id}/client-portal` 별도 1분 점검 |
